#  Projeto BI - Academia de Jiu-Jitsu

Projeto de Business Intelligence desenvolvido no Power BI com foco em modelagem dimensional, ETL e construção de dashboards gerenciais.

---

##  Objetivo

Desenvolver um dashboard analítico para acompanhamento de indicadores de desempenho de uma academia de Jiu-Jitsu, permitindo análise estratégica e tomada de decisão baseada em dados.

---

##  Tecnologias Utilizadas

- Power BI
- Modelagem Dimensional (Star Schema)
- ETL
- DAX
- Excel (base de dados)
